## 摘要

这本学习指南介绍了如何使用Yeoman从头开始构建一个Todo应用程序，从而了解如何搭建一个项目的脚手架。
在构建的过程中将会用到[Grunt](http://gruntjs.com/)，[Bower](http://bower.io/)以及[AngularJS](https://angularjs.org/)。
